1) Instale a extenção do Jupyter Notebook
2) Crie o ambiente virtual com o seguinte comando: python -m venv env
3) Adicione o "./env/Scripts/python" no arquivo 'main.ipynb' (botão para selecionar o env ao lado direito superior)
4) Acione o ambiente virtual através do terminal com o comando: source env/Scripts/activate
5) Instala as dependencias com o comando: pip install -r requirements.txt
6) Rode os códigos no arquivo 'main.ipynb'
